Tentang Saya

Saya adalah mahasiswa jurusan Ilmu Komputer yang memiliki minat besar dalam Pengembangan Web. Saya memiliki pengetahuan dalam HTML, CSS, JavaScript, Kotlin dan framework modern seperti Laravel dan Tailwind. Saya sedang mencari kesempatan magang sebagai Web Developer dan/atau Mobile Developer untuk memperdalam keterampilan dan mendapatkan pengalaman profesional.

Saya suka memecahkan masalah dan menciptakan antarmuka pengguna yang menarik secara visual dan berfungsi dengan baik. Saya terus belajar teknologi baru dan mengikuti tren terbaru dalam pengembangan web.

Beberapa kualitas yang saya miliki:

  • Keterampilan pemecahan masalah yang kuat
  • Perhatian terhadap detail
  • Kemampuan bekerja dalam tim
  • Komunikasi yang efektif
